# Atajos de teclado para TimeMeddler

| Atajo         | Descripción                                                                           |
| ------------- | --------------------------------------------------------------------------------------|
| Ctrl+T  		| inicia la actividad de tipo Trabajo	|
| Ctrl+D		| inicia la actividad de tipo Descanso |
| Ctrl+P		| inicia la actividad de tipo Procrastinación		|
| Ctrl+S		| activa el botón Stop		|
